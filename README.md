## 星星（★）邏輯——非常重要
★ = 跳空漲停（一價到底）：**開盤 ≈ 漲停價 且 最低 ≈ 漲停價**
- 漲停價計算：`round(前日收盤 × 1.1, 2)`
- 容差：`< 0.02`（因為價格級距取整會有誤差）
- **不是**只要跳空高開就給星
- **不是**只要收盤漲停就給星
- 必須全天沒有低於漲停價的成交，才算一價到底

## 星星套用範圍
- 二手紅盤（s1）：判斷第1天、第2天，共2天
- 三手紅盤（s4）：判斷第1、2、3天，共3天
- 四手紅盤（s5）：判斷第1、2、3、4天，共4天
- 五手紅盤（s6）：判斷第1、2、3、4、5天，共5天（全部都判斷）
- 跌停翻漲停（s2）、其他策略：無星星

## FinLab 使用欄位
- `price:收盤價`
- `price:開盤價`
- `price:最低價`（用於判斷一價到底）
- `company_basic_info`（公司名稱、產業）
